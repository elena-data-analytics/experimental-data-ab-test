# Statistical Analysis of Experimental Data (A/B Case)

## Описание проекта
Проект представляет собой учебно-аналитический кейс, выполненный на синтетических экспериментальных данных. Цель проекта — продемонстрировать подход к анализу экспериментальных данных, проверке статистических гипотез и корректной интерпретации результатов.

## Аналитический вопрос
Существует ли статистически значимое различие между экспериментальными группами A и B по биологическому показателю объёма вирусной ткани в организме (метрика VIRUS)?

## Данные
- Тип данных: синтетические экспериментальные данные  
- Экспериментальные группы:  
  - Группа A — воздействие G-вируса  
  - Группа B — воздействие T-вируса  
- Дополнительные признаки: город, гендер, физиологический показатель PULSE  

Данные используются исключительно в учебно-методических целях и не отражают реальный биологический или бизнес-процесс.

## Методология анализа
- предварительный анализ распределений и проверка допущений;
- сравнение экспериментальных групп с использованием статистических тестов;
- описательный стратифицированный анализ по городам и гендеру;
- корреляционный анализ между показателями VIRUS и PULSE;
- интерпретация результатов с учётом силы эффекта и ограничений данных.

## Основные результаты
- Статистически значимых различий между группами A и B по показателю VIRUS на уровне всей выборки не выявлено (p-value ≥ 0.05), несмотря на небольшие различия средних значений.
- Стратифицированный анализ по городам и гендеру не выявил устойчивых статистически подтверждённых различий; результаты в данных срезах носят описательный характер.
- Между показателем VIRUS и физиологическим показателем PULSE обнаружена статистически значимая, но слабая линейная связь на уровне всей выборки.
- В группе B наблюдается более выраженная, однако всё ещё ограниченная по силе корреляция между VIRUS и PULSE; в группе A значимая связь не выявлена.

## Ограничения анализа
- данные являются синтетическими и симуляционными;
- анализ носит учебно-методический и описательный характер;
- причинно-следственные выводы не делаются;
- результаты валидны только в рамках заданных условий эксперимента.
